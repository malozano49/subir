Simple Search

Simple search is the default search on the homepage.

Filters that can be applied to a search

When performing a search, you can consult all available sources or a specific one.

By default, the search is conducted in all available sources. If you want the search to be carried out in a specific source, select it:

In addition to being able to select the sources for the search, ODA allows you to narrow down the search to a specific set of records, which we will call subcatalogs.

  • You can select a subcatalog from the list of subcatalogs or locate it by entering certain characters.

How to Search

Before you start searching, please consider the following points:

  • Search terms can be entered in any way, including uppercase or lowercase letters and diacritics, accents, umlauts, etc., or without them, etc., or without them.
  • Any term entered is considered a search term: a word, a number, a string of characters, etc.
  • There are no defined stop words. Prepositions, articles, etc. are included as searchable to facilitate the exact location of documents.
  • When multiple search terms are entered in the same query, they can be combined with an operator that indicates their relationship in each document. The meaning of each of the operators that can be used is as follows:
Operator Description
AND (&) Locates records that contain all the terms connected with this operator.
OR (|) Locates records that contain any of the terms connected with this operator or both.
NOT Locates records that contain the first term connected with this operator but not the second.

  • You can perform exact searches by enclosing all the terms in double quotes (").
    This way, you will locate all records that contain the text entered within quotes in the same order as entered in the search.
  • You can use wildcards to locate parts of words. Wildcards can be of two types:
Description
? Placed in any position in the search term, each ? replaces a single character. This way, you will locate all documents that meet the search pattern and that contain any character where the ? sign is located.
* Placed in any position of the term, or in several at once, it replaces a string of characters. The asterisk * will locate all documents that meet the search pattern and that contain any number of characters where the * sign is located.

You can perform the following searches:

  • - By a single term
  • - By multiple terms
  • Exact searches
Search by a Single Term
1. Enter the search term you want to locate.
2. Click to locate the records containing the entered term.

Remember that you can also perform searches using wildcards
Search by Multiple Terms
1. Enter the search terms you want to locate connected with one of the operators explained earlier.
2. Click to locate the records containing the entered terms.

Remember that you can also perform searches using wildcards
Exact Searches

This search retrieves records that contain the exact expression searched for as entered.

1. Enter the search terms you want to locate enclosed in double quotes.
2. Click to locate the records containing the exact expression entered.

Remember that you can also perform searches using wildcards

"Did you mean"

If no results are found in the search or fewer than 10 records are located, ODA suggests terms to users that may help them find records of interest based on spellchecking.

Spellchecking is a spelling check that involves reviewing the query entered by the user and suggesting an alternative query that is orthographically correct.

Authority Search

If no results are found in the search or fewer than 10 records are located, ODA performs a search in the authority database to see if it can locate any authority that matches the entered search.

Access to Authorities Search

To perform a search for authorities to locate records containing a specific authority, click on the authorities search option.

Access to Index Search

To perform a search that locates words or patterns of words similar to the entered term or terms, click on the index search option.

Access to Recent Searches

To view the 10 most recent searches that have yielded results during the OPAC session, click on recent searches.

Access to Saved Searches

If you want to save any of the searches you have performed to be able to run them at another time, click on saved searches.

  • To save a search, it is necessary to identify yourself beforehand.

Access to Favorites

To view the records you have marked as favorites, click on Favorites.

Access to My Profile

Click on the My profile option and identify yourself to access the following ODA utilities:

  • Personal information
  • Requests
  • Active loans
  • Loan history
  • Renew loans
  • Cancel reservations
  • Save search
  • Create favorites lists

To identify yourself as a user, you can use any of these 3 methods:

  • Library card number
  • Email provided to your library
  • Mobile phone provided to your library
