How to Search

Before starting a search, please consider the following points:

  • Search terms can be entered in any way, whether in uppercase or lowercase, with diacritics, accents, umlauts, etc., or without them.
  • Any term entered is considered a search term: a word, a number, a string of characters, etc.
  • There are no defined empty words. Prepositions, articles, etc., are kept searchable to facilitate the exact location of documents.
  • When multiple search terms are entered in the same query, they can be combined with an operator that indicates their relationship in each of the documents. The meaning of each of the operators that can be used is as follows:
    Operator Description
    AND (&) Locates records containing all terms combined with this operator.
    OR (|) Locates records containing any of the terms combined with this operator or both.
    NOT Locates records containing the first term combined with this operator but not the second.
  • Exact searches can be performed by enclosing all terms in double quotes (").

    This way, all records containing the text entered within double quotes in the same order as entered in the search are located.

  • Truncation can be used to locate parts of words. Truncation can be of two types:
    Description
    ? Placed at any position in the search term, each ? replaces a single character. This way, all documents that match the search pattern and contain any character in the place where the ? sign is located are located.
    * Placed at any position in the term, or in multiple positions at once, it replaces a string of characters. The asterisk * will locate all documents that match the search pattern and contain any number of characters in the place where the * sign is located.

    The following types of searches can be performed:

Searches in All Fields

By default, advanced search is performed in Any Field.

The search you perform will consider all fields in the database that are defined as searchable.

Searches in a Specific Field

To perform a search in a specific field, select the one you are interested in from the Field option. By default, the fields in which the search can be performed are as follows:

The search you perform will only consider the selected field.

Search by Term

1. Enter the search term you want to locate in the Term field to search.

2. Click on to locate the records that contain the entered term.

Remember that you can also perform searches using truncation.

Search by Multiple Terms

1. Enter the search terms you want to locate in the Term field to search

2. Click on to locate the records that contain the entered terms.

Remember that you can also perform searches using truncation.

Searches Containing All Entered Terms

By default, advanced search locates all entered search terms.

Searches Containing Some or All Entered Terms

If you want the search to only locate some or all of the entered search terms, select the Any Word option.

Exact Searches

If you want the search to retrieve records that contain the searched expression exactly as entered, select the Exact Phrase option.

Searches Using Operators

The operators that can be used in advanced search are as follows:

  • The default operator used in advanced search is AND.
  • The search by operators is displayed when using the option to combine multiple searches.
  • If you want to remove any of the added combinations, click on the option associated with the one you don't want to apply.

Searches Using Filters

By default, the following filters can be used in advanced search:

  • Language
  • Country
  • Medium
  • Document Type

Search by Publication Year

By default, advanced search enables searching by publication year.

How to Search for Records Published in a Specific Year
To locate records published in a specific year, enter the desired year in the Start Year and End Year fields.

How to Search for Records Published from a Certain Year
To locate records published from a specific year, enter the desired year in the Start Year field.

How to Search for Records Published Before a Certain Year
To locate records published before a specific year, enter the desired year in the End Year field.

How to Search for Records Published in a Specific Period
To locate records published in a specific period,
  • 1. Enter the first year in the Start Year field.
  • 2. Enter the last year in the End Year field.

Automatic Search Suggestions

ODA provides the following possibilities for automatic search suggestions:

  • 1. "Did you mean"
  • 2. Search in authorities
"Did you mean"

If no results were found in the search or fewer than 10 records were located, ODA suggests terms to users that may help them obtain records that may be of interest based on spellchecking.

Spellchecking is a spell-check that involves checking the query entered by the user and suggesting an alternative query that is orthographically correct.

Example:
  • The term "estilmismo" is entered
  • Since the search did not return any results, it suggests the following terms as recommendations:
  • If you click on any of the suggested terms, it performs a query to locate records containing the selected term
Search in Authorities

If no results were found in the search or fewer than 10 records were located, ODA conducts a search in the authorities database to see if it can locate any authorities that match the entered search.

Access to Simple Search

To access simple search, click on the Simple Search option

Access to Authority Search

To perform an authority search that allows you to locate records containing a specific authority, click on the Authority Search option.

Access to Index Search

To perform a search that locates words or word patterns similar to the entered term or terms, click on the Index Search option.

Access to Last Searches Conducted

To view the 10 most recent searches that have returned results during the OPAC session, click on Last Searches.

Access to Saved Searches

If you want to save any of the searches you have conducted to be able to run them at another time, click on Saved Searches.

  • To save searches, it is necessary to identify yourself beforehand

Access to Favorites

To view the records you have marked as favorites, click on Favorites.
