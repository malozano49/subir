Authority Search

The Authority Search option allows you to perform an authority search to facilitate the location of records containing a specific authority.

Types of Searches

The following types of searches can be performed:

  • - Searches in all fields
  • - Searches in a specific field
  • - Searches containing the entered term or terms
  • - Searches starting with
  • - Exact searches

Searches in all fields

By default, the search is performed in All Fields.

The search you perform takes into account all fields in the database that are defined as searchable.

Searches in a specific field

To perform a search in a specific field, select the one you are interested in from the Field Selection option. By default, the fields available for searching are as follows:

The search you perform will only consider the selected field.

Searches containing the entered term or terms

To locate authorities containing one or more search terms, use the Contains in Selection Filters option.

The search is performed using internal truncations in front of and behind the entered search term. To refine the search, you can use the term in quotes.

Example:

  • If you enter the term "sagrario," the search performed is: *sagra*. In this way, it will locate authorities such as the following:
    • Cantalejo Ruiz, Sagrario
    • Fierro Madrid, Sagrario
    • Ib치침ez, Sagra
  • If you enter "sagra," it will only locate those containing the term "sagra":
    • Ib치침ez, Sagra

Searches Starting With

To locate authorities that start with a specific term, select the Starts with in Selection Filters option.

The "Starts with" option is only available if a search field has been selected previously.

Exact Searches

To locate authorities that match exactly with the search string entered, select the Exact in Selection Filters option.

The "Exact" option is only available if a search field has been selected previously.

How to Sort the Search Results

The authorities found in a search are sorted by default in alphabetical order. If you wish, you can change the sorting based on relevance.

How to Navigate the List of Authorities Found in the Search

By default, authorities found in a search are displayed in groups of 10, but there is an option to group the results by 25 or 35.

To navigate through the different pages of the results obtained, you have the following options:

  • Click on to move to the next page.
  • Click on to move to the previous page.
  • Click on the page number you want to view
  • * Click on the number 1 to view the first page

  • Click on the number before "Next" to go to the last page

How to Find Records Containing a Specific Authority

Once you have made the query you are interested in authorities, a list of authorities corresponding to the entered search is displayed, along with the number of records containing the located authorities.

Access to Index Search

To perform a search that locates words or word patterns similar to the entered term or terms, click on the Index Search option.

Access to Recent Searches

To view the 10 most recent searches that have returned results during the OPAC session, click on Recent Searches.

Access to Saved Searches

If you want to save some of the searches you have conducted to run them at another time, click on Saved Searches.

  • To save searches, you need to identify yourself beforehand.

Access to Favorites

To view the records you have marked as favorites, click on Favorites.

Access to My Profile

Click on the My Profile option and log in to access the following ODA utilities:

  • Personal Information
  • Requests
  • Active Loans
  • Loan History
  • Renew Loans
  • Cancel Reservations
  • Save Searches
  • Create Favorites Lists

To identify yourself as a user, you can use any of these 3 methods:

  • Library card number
  • Email address provided to your library
  • Mobile phone number provided to your library

Access to the Home Page

To go to the home page, click on Home.
