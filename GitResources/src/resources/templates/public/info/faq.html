Preguntas frecuentes [faq.html]

Si al buscar, dado un texto de búsqueda, aparece un diálogo rojo con el mensaje: El texto de búsqueda está mal formado. Asegúrate que no contiene caracteres reservados. nos está indicando que existe uno o varios caracteres inválidos en el texto de búsqueda.

Si estamos buscando un texto exacto, debemos entrecomillar el texto de búsqueda o parte de este, para encontrar resultados. Los caracteres ( ) [ ] : ~ suelen tener un significado concreto para la búsqueda, por lo que entrecomillar las palabras que contengan estos caracteres, suele ser una solución.
