Clase unlogged

Aplicación de .unlogged La clase .unlogged se puede aplicar sobre un contenedor padre y muestra un ángulo de color sobre los distintos componentes clickables contenidos. Para evitar conflictos en estilos con fontawesome, se aplica sobre el padre de los iconos (por ejemplo en .btn, como padre de un span titulo + un span .icono)
Vista de clase 
.unlogged{
  .btn{@include unlogged(red);}
  .nav-link{@include unlogged(green);}
  .link{@include unlogged(DARKORANGE);}
  .page-link{@include unlogged(blue);}
  .form-group{
    &.cont_input{@include unlogged(red);}
    &.cont_selectPicker{
      .btn{
        &:before{display:none}
      }
      @include unlogged(red);
    }
  }
}
