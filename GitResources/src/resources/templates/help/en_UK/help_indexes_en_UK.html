Index Search

The Index Search option allows you to perform a search to locate words or patterns of words similar to the entered term or terms.

The results located in the search are sorted by default in alphabetical order.

Types of Searches

You can perform the following types of searches:

  • Searches in a specific field
  • Searches in all fields
  • Searches containing terms similar to the search term entered
  • Searches starting with

a. Searches in a specific field

By default, the index search is performed in the Title field. To perform a search in any other field, select the one you are interested in from the Field Selection option. By default, the fields in which the search can be performed are as follows:

The search you perform will only consider the selected field.

b. Searches in all fields

If you want the search to be performed in any field, select the All Fields option.

The search you perform will consider all fields in the database defined as searchable.

c. Searches containing terms similar to the search term entered

To locate in the index records containing terms similar to the entered term, use the Contains in Filter Selection option.

d. Searches that start with

To locate in the index records that start with terms in the search string entered, select the Starts with in Filter Selection option.

Navigating the List of Results Located in the Search

By default, the search result is displayed in groups of 10, but you have the option to group the results in sets of 25 or 35.

To navigate through different groups,

  • Click on to move to the next page.
  • Click on to move to the previous page.
  • Click on the page number you want to view
  • Click on the number 1 to view the first page
  • Click on the number before Next to go to the last page

Locating Records Associated with Search Results in the Index

Once you have performed the query of interest in the index, a list of authorities corresponding to the entered search is displayed, along with the number of records containing the located terms.

Access to Authorities Search

To perform a search for authorities that allows you to locate records containing a specific authority, click on the Authorities Search option.

Access to Recent Searches

To view the 10 most recent searches that have yielded results during the OPAC session, click on Recent Searches.

Access to Saved Searches

If you want to save any of the searches you have performed to be able to run them at another time, click on Saved Searches.

  • To save searches, it is necessary to identify yourself beforehand.

Access to Favorites

To view the records you have marked as favorites, click on Favorites.

Access to My Profile

Click on the My Profile option and identify yourself to access the following ODA utilities:

  • Personal information
  • Requests
  • Active loans
  • Loan history
  • Renew loans
  • Cancel reservations
  • Save searches
  • Create favorites lists

To identify yourself as a user, you can use any of these 3 methods:

  • Library card number
  • Email provided to your library
  • Mobile phone provided to your library

Access to Homepage

To go to the homepage, click on Home.
